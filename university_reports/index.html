985/C9与国科大推免专业与导师分类报告

按学校和科研平台拆分的 HTML 报告入口。每份报告延续东南大学报告结构，包含适合专业、推荐导师、研究领域、代表成果、自我介绍话术、准备清单和资料来源。

资料时点：2026-06-26 适用背景：西电电子信息大类 / 遥测遥感 学校数量：14

学校报告入口

清华大学

综合排名冲刺

清华适合成绩排名、科研经历和英语表达都很强的学生。主线不要泛泛写通信，而应强调“西电电子信息基础 + 雷达/SAR 或空间网络 + 空天信息处理”。

主推导师：杨健、陆建华、牛志升

北京大学

C9综合排名冲刺

北大适合两条路线：遥感与地理信息研究所的遥感图像/SAR/GIS，以及电子学院的信息与通信工程、无线通信和信号处理。整体难度极高，更适合成绩排名和科研经历都突出的申请人。

主推导师：梁存任、李培军、宋令阳

上海交通大学

综合排名冲刺

上交适合通信、6G、通感一体、低空网络、图像通信和导航感知方向。对你来说，它比纯遥感更适合走“电子信息 + 空天/低空应用”的叙事。

主推导师：陶梅霞、武庆庆、郁文贤

复旦大学

C9综合排名冲刺

复旦适合通信、6G、智能无线网络、雷达遥感和图像处理交叉方向。相比武大、北理工，它不是最传统的遥感强校，但若你想冲 C9 综合平台并保持电子信息主线，可以作为冲刺选项。

主推导师：徐丰、朱宇、杨涛、王斌

浙江大学

综合排名冲刺

浙大适合综合平台冲刺。通信方向更主线，遥感方向偏环境资源和农业遥感；如果你有高光谱、植被、灾害遥感项目，匹配度会上升。

主推导师：蔡云龙、黄敬峰

武汉大学

遥感强匹配

武大是遥感、测绘、摄影测量和遥感智能处理的强匹配学校。如果你的遥感图像处理、机器学习、GIS、摄影测量项目更突出，武大优先级不低于综合排名冲刺校。

主推导师：张良培、钟燕飞、张永军

北京理工大学

雷达/SAR强匹配

北理工非常适合雷达、SAR、空天遥感、目标识别方向。它的匹配度取决于你能否把“遥测遥感”讲成“雷达探测、SAR 成像和智能解译”。

主推导师：曾涛、师皓、李涵

电子科技大学

电子信息强匹配

电子科大和西电背景最容易讲通，适合雷达成像、无线通信、抗干扰通信、6G、多域传输。若你想提高命中率，它应作为强匹配学校重点准备。

主推导师：杨建宇、杨平

北京航空航天大学

空天信息强匹配

北航适合空天地一体化、低空经济、无人系统、导航监视、跨域通信和多源感知。它比传统遥感更偏空天系统与工程应用。

主推导师：付进、张军

中国科学技术大学

C9冲刺 / 理工强校

中科大适合雷达信号处理、毫米波感知、图像视频处理、AI 地球物理和遥感信号处理。它的风格偏基础研究和硬核理工，需要你把数学、信号处理和科研训练能力讲清楚。

主推导师：陈卫东、伍新明、刘东、陈力

南京大学

C9冲刺 / 遥感通信双线

南大适合两条线：电子学院的通信、空天地一体化网络、智能通信；地理与海洋科学学院的遥感图像处理、GIS 和多源遥感融合。

主推导师：周海波、杨鲲、杜培军、肖鹏峰

哈尔滨工业大学

C9 / 空天雷达强匹配

哈工大非常适合雷达、空天地通信、空间智能遥感、SAR/ISAR 成像和目标识别。它的航天、国防和工程应用属性强，与你的电子信息和遥测遥感背景贴合度高。

主推导师：孟维晓、谷延锋、吴小川、张钧萍

西安交通大学

C9 / 西北强校

西交大适合信息与通信工程、通信感知一体化、遥感光电探测、空间智能和信号处理方向。你在西安读本科，地域联系和院校交流便利也是优势。

主推导师：郑通兴、张淳民、曹相湧、杨在

中国科学院大学

中科院体系冲刺 / 科研院所强匹配

中国科学院大学不是传统 985，但依托中科院研究所体系，科研训练强度和导师资源非常强。对你来说，国科大最适合走“电子信息 + 遥测遥感 + SAR/微波成像/高光谱 + 中科院科研平台”的路线，也可以保留通信与信号处理备选。

主推导师：吴一戎、郭华东、张兵、高绍帅

使用建议

优先级：清华、北大、上交、复旦、浙大、中科大、南京大学、哈工大、西交大属于综合平台冲刺；中国科学院大学属于中科院体系科研平台冲刺；武汉大学、北理工、电子科大、北航属于与你方向强匹配的专业冲刺。

申请策略：不要只按学校名气排序。你的最强叙事是“电子信息基础 + 信号处理/通信 + 雷达/SAR/遥感智能处理 + 空天应用场景”。
