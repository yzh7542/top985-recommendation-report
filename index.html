985 Recommendation Report

面向西电电子信息大类学生的 985 高校推免专业与导师推荐报告

适用背景：西安电子科技大学大三学生，本科方向为信息力学与遥测遥感，属于电子信息大类；目标为保送综合排名靠前且电子信息、通信、雷达遥感、空天信息方向实力强的 985 高校。

资料时点：2026-06-26 目标：推免 / 夏令营 / 预联系 核心方向：通信 + 雷达/SAR + 遥感智能处理 来源：高校官网、学院/导师主页、官方新闻

一页结论

最建议的申请主线：不要只按“遥测遥感”去投传统测绘，也不要只按“通信”去投纯通信。你更有竞争力的交叉叙事是：电子信息基础 + 信号处理/电磁与雷达 + 遥感图像智能处理 + 通信感知一体化。

冲刺顺序建议：清华大学、上海交通大学、浙江大学作为综合排名冲刺；武汉大学、北京理工大学、电子科技大学、北京航空航天大学作为专业强匹配冲刺。若个人成绩排名非常靠前且已有科研/竞赛成果，可把清华和上交放在第一批预联系；若遥感项目经历更强，武汉大学和北理工雷达方向应优先联系。

  • 清华、上交、浙大：综合排名冲刺，要求成绩排名、科研和英语表达都要强。
  • 武大、北理工、电子科大、北航：专业匹配更强，若项目经历贴合，成功率不一定低于综合名校。

申请材料包装建议

你的主线叙事

建议把自己包装成“具备电子信息、信号处理和遥测遥感复合背景，愿意做空天信息智能感知、雷达遥感或通信感知一体化研究”的学生。这样既能解释为什么从西电电子信息大类申请通信/电子信息，也能解释为什么申请遥感或 SAR 方向。

简历重点

  • 成绩：写明专业排名、绩点、核心课成绩，突出信号与系统、数字信号处理、通信原理、电磁场与微波、随机信号、机器学习、遥感图像处理、测量/导航相关课程。
  • 项目：至少准备一个可展示项目，例如 SAR 图像目标检测、遥感变化检测、高光谱分类、InSAR 形变监测、OFDM 通感一体仿真、低空/卫星通信链路建模。
  • 代码与文档：用 GitHub 或 PDF 展示实验流程、数据来源、指标、可视化结果。只写“做过遥感项目”不够，要证明你会处理数据和复现实验。
  • 联系邮件：不要群发同一封。每位导师至少写 2-3 句与其方向相关的匹配理由。

  • 本报告按 2026-06-26 可查公开资料制作。2027 届推免时，各校招生专业、名额和导师招生状态可能变化，必须以当年学院夏令营、预推免、推免目录和导师组通知为准。
